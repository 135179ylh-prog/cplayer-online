# PRD - cplayer-online

## 目标
把 ChKSz CPlayer 改造成手机可用的在线音乐播放器。

## 已完成
- 后台播放修复（去除 WebAudio 路由）
- 播放列表持久化（IndexedDB）
- 自建歌单（新建/加入/播放/删除）
- 4 种播放模式（顺序/单曲/列表循环/随机）
- 手机端布局适配（华为 Pura X 阔折叠）
- 搜索后列表不消失
- 歌单内歌曲排序（管理详情、上移/下移、移除与顺序持久化）
- 最近播放列表（真实播放记录、去重、50 首上限、重播与清空）
- 自建歌单 JSON 备份导出/原子导入
- 播放失败按模式有界自动跳过
- 搜索连续收藏：加入歌单后保留搜索词、结果和桌面/移动面板
- 歌单操作清晰化：当前播放“歌单”入口、资料库文字操作按钮
- 移动搜索动作清晰化：直接显示“加入”和“歌单”
- 跨网络访问（GitHub Pages）

## 待完成
- 当前产品清单已完成；后续按实际使用反馈继续维护。

## 2026-07-19 稳定性加固
- API 超时与搜索结果竞态保护
- 队列写入串行化、页面生命周期冲刷和空队列恢复
- JSON 队列导入规范化与移动端同步
- Service Worker 更新回退、核心资源保护、离线壳隔离和封面缓存上限
- 默认 playlist.js 入口与外部歌曲文本安全渲染
